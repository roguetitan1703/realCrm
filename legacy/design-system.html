Bhumi Propcity — Design System

Three subject-grounded directions for a Pune brokerage CRM. Pick one (or mix), then it applies across every screen. Every component shown with real product content.

How to read this: Each block is a complete visual world — palette, type pairing, then the shared component set (buttons, pills, status system, cards in card and list form, the one filter/sort bar, inputs, the follow-up scheduler, toasts, side-nav, action rail). The components are identical in structure across all three — only the skin changes — which is exactly the coherence goal: one grammar, one identity.

Direction A

Registry
The gravitas of a sub-registrar's office and stamp-paper. Ink-teal ground, gold-leaf money accent, document-paper surfaces. Fraunces (a warm, high-character serif) for headings and figures reads as institutional and trustworthy — like a firm that's been registering deals for 30 years. Body in Instrument Sans stays clean and modern so it never feels dated.

Direction B

SALE BOARD
The graphic energy of Indian property signage — bold, high-contrast, unapologetically informational. Graphite ground with a confident signboard amber. Archivo (a chunky grotesque) for prices and headings hits like a printed board; Spline Sans keeps data legible. Near-zero ornament, colour does the work. More energetic and modern than A — reads like a sharp young firm.

Nameplate (refined)
