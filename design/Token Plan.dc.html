05 — Layout concept
Mobile-first "field tool", desktop widens to "back-office". Every screen wears a slim bottle-green identity bar (the nameplate), porcelain body, white cards keylined in sage. Money and success carry a single brass hairline/figure — nothing else gets the metal. Prices are set in Space Grotesk tabular, Indian-formatted (₹82L, ₹1.2Cr), as first-class typographic objects. Dense rows over decoration: a lead row shows name, requirement one-liner, source chip, stage dot and time-ago at a glance. Bottom tab bar on mobile (Home / Leads / Properties / Contacts / More); left rail on desktop admin. One orchestrated motion moment (3.8); everything else is quiet micro-interaction. All values above live in theme.js — tenant #2 is a config swap.
